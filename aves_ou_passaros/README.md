# Aves ou Pássaros?

- Desenvolvimento de árvore de decisão que procura por meio da seleção de informações verificar se uma ave é ou não um pássaro.

- Ferramentas utilizadas: `HTML`, `CSS`, `JavaScript`.

- IDE: `Visual Studio Code`.
